# Food Service

This is a lightweight service for making food orders.

## Installation

```bash 
 # install packages
$ npm install

# start docker for postgres
$ docker-compose up -d
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```
